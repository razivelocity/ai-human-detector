# AI vs Human Text Detector 📝🤖

This project is a **Flask web application** that predicts whether a given text is written by a **Human** or generated by an **AI model**.  
It uses a combination of **TF-IDF text features** and **linguistic metadata** (e.g., perplexity, burstiness, coherence) from the dataset `AuthentiText_X_2026_AI_vs_Human_Detection_1K.csv`.
